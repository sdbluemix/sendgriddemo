sendgriddemo
============

SendGrid service usage demo in IBM Bluemix. 
If you want to download and use the code in your application , you are free to do so. SendGridDemo folder is an Ecplise project. In case any compilation error occurs due unbound class patha variables, then change the JDK/JRE to JDK/JRE of your choice ( >=V1.6) and you may replace IBM Bluemix Runtime with and other web project runtime available). Code should be running without any issues.
